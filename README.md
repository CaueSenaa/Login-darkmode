# 🔐 Login Dark Mode

Projeto de interface de autenticação desenvolvido com React + CSS moderno, contendo sistema de login e cadastro com dark mode, validação de formulário e responsividade.

---

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS3
- Google Fonts
- Material Symbols

---

## ✨ Funcionalidades

✅ Tela de login  
✅ Tela de cadastro  
✅ Dark Mode / Light Mode  
✅ Mostrar e ocultar senha  
✅ Validação de e-mail  
✅ Verificação de confirmação de senha  
✅ Layout responsivo  
✅ Interface moderna  
✅ Animações suaves  
✅ Ícones integrados  

---

## 📸 Preview

O projeto possui:
- Card centralizado
- Alternância entre login e cadastro
- Inputs com ícones
- Botões modernos
- Interface inspirada em aplicações reais
